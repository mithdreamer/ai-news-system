# AI Haber Yayın Sistemi

## Amaç

RSS kaynaklarından haber çekmek,
haberleri filtrelemek,
kategorize etmek,
istatistik üretmek,
HTML ve TXT formatında yayınlamak.

## Özellikler

- RSS Haber Çekme
- Haber Temizleme
- Haber Filtreleme
- Kategori Sistemi
- İstatistik Üretimi
- HTML Çıktı
- TXT Çıktı
- Loglama
- Tarih Bazlı Arşivleme

## Gelecek Özellikler

- MP3 Üretimi
- Video Üretimi
- Web Arayüzü
- YouTube Otomasyonu
